Ausführliche Beschreibung

Das Hauptmenü des MainWindow.

Verwaltet alle möglichten Actions in einer QMenuBar.
Dabei wird (zumindest unter OS X) darauf geachtet, dass die Icons von den Aktionen entfernt werden.
