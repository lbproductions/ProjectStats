Ausführliche Beschreibung

Der Startpunkt dieses Programms.

Hier wird sichergestellt, dass immer nur eine Datenbank und ein MainWindow gleichzeitig geöffnet sind. Außerdem wird einfacher Zugriff auf die aktuelle Datenbank gegeben.
