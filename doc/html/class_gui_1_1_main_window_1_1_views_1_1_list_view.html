Dieser Slot wird aufgerufen, wenn ein Spiel in der ItemsList doppelt geklickt wurde.
Dann wird ein neues zum Spiel passendes Fenster geöffnet (z.B. ein LiveGameWidget)

void ListView::on_rowList_rowsSelected ( QList< Database::Row * >  list) [private, slot]

Dieser Slot wird aufgerufen, wenn in der ItemsList ein Spiel ausgewählt wurde.
Dann wird das zum Spiel passende Widget als DetailsWidget gesetzt.

void ListView::on_rowList_selectionChanged ( ) [private, slot]

Dieser Slot wird aufgerufen, wenn sich in der ItemsList die Auswahl geändert hat.
Nach dieser sehr allgemeinen Info wird das DetailsWidget invisible gesetzt, was flackern verhindert, beim setzen des nächsten Widgets.
